Syntax

WowzaProvider

Summary

Parameters:

  • providerConf Object

    A wowza configuration object

    • host String

      Server host

    • user String

      Wowza user

    • pwd String

      Wowza user password

    • [protocol=ftp] String optional

      Server protocol (ftp, frtp, sftp or ftps), protocol is added on beginning of host, ex : sftp://domain.com in this case

    • [port] Number optional

      Server port added to the end of the host, ex: sftp://domain.com:22 in this case
