(private, static) resolveResourcesUrls(medias)

Resolves medias resources urls using CDN url.

Medias may have attached resources like files associated to tags, timecodes images, thumbnail image and so on. These resources must be accessible through an url. As all resources must, in the future, reside in a CDN, resolveResourcesUrls transforms all resources URIs to URLs based on CDN.
